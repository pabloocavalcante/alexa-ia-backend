# 🧠 Alexa IA Backend

Backend para integração da Amazon Alexa com uma IA avançada (OpenAI).

## 🚀 Objetivo

Permitir que a Alexa responda perguntas de forma inteligente usando IA, superando as limitações padrão.

---

## ⚙️ Como funciona

Fluxo da aplicação:

Usuário → Alexa → Skill → Backend → OpenAI → Resposta → Alexa

---

## 🛠️ Tecnologias utilizadas

- Node.js
- Express
- OpenAI API

---
